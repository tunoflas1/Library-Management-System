📘 Library Management System
📌 Project Description

This project is a Library Management System developed in Python using Object-Oriented Programming (OOP).
It allows users to manage books, members, and loan operations through a menu-driven interface.

The system provides basic functionalities such as:

Adding and managing books
Registering and managing members
Borrowing and returning books
Tracking active and overdue loans
Saving and loading data using CSV files
⚙️ Features
📚 Book Management
Add new books
Search books by title or author
Update book genre
List all or available books
👤 Member Management
Register new members
View member details
Track active members
🔄 Loan Management
Borrow books
Return books
Track active loans
Detect overdue loans
💾 Data Persistence
Data is saved in CSV files
Automatically loaded when the program starts
🧠 Technologies Used
Python
OOP (Object-Oriented Programming)
CSV file handling
Exception handling

Go to project folder:

Run the program:
